Required change

Add a relevant visual anchor to the hero section.

Acceptable options

Implement one of the following:

Task 6: Add two missing high-friction FAQ items
Problem

The FAQ is good, but it still misses two common buyer objections for data-only eSIM products.

Required change

Add these FAQ items:

Question:
O meu número de WhatsApp muda?

Answer:
Não. Na maioria dos casos, podes continuar a usar o teu número normal no WhatsApp, desde que a tua conta já esteja ativa no telemóvel.

Question:
Posso usar o meu SIM atual e o eSIM ao mesmo tempo?

Answer:
Em muitos telemóveis compatíveis, sim. Podes manter o teu SIM físico para chamadas e usar o eSIM para dados móveis, dependendo do equipamento.

Acceptance criteria

both items are added to the FAQ accordion

accordion remains readable and performant

ordering is logical with the existing FAQ items
